WebChatting
===========

A simple WebIM demo based on WebSocket.


笔者测试环境：
  服务器：Tomcat7.0.42
  IDE   : Eclipse JUNO
  
  
说明：
  
  这是一个基于WebScoket协议的简易网页聊天示例，由于WebScoket为HTML5新增功能，所以在进行测试时，需选用支持HTML5的浏览器。

  该示例的基本功能为，用户进入聊天窗口时，需输入一个用户名（未进行任何检查）以用作聊天时在线显示。页面较为简单，直接点
击链接即可。笔者在客户端网页界面里用JavaScript进行链接操作，配置的为localhost URI，非本地模式时，需要自行修改其URI。
